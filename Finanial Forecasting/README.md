# Financial Forecasting

This repository contains a simple Java financial forecasting example.

## Files

- `FinancialForecast.java` - sample program that projects revenue, expense, and profit for 12 months.

## Run

Compile and run the program from the command line:

```bash
javac FinancialForecast.java
java FinancialForecast
```

## Description

The example starts with an initial revenue and expense, applies a monthly growth rate, and prints monthly projected revenue, expense, and profit.
