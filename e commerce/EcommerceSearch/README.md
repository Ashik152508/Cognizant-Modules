# EcommerceSearch

A simple Java project for managing product search, filtering, and sorting in an ecommerce-like system.

## Files

- `Product.java` - Represents a product with id, name, category, price, and rating.
- `SearchSystem.java` - Provides search, category filtering, and sorting functionality.

## Usage

Compile:

```
javac Product.java SearchSystem.java
```

Run:

```
java SearchSystem
```

Note: `SearchSystem` is designed as a reusable class and can be integrated into a larger application.
